# Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/usuario/chatlearn.git
   cd chatlearn

2. Configure o ambiente:

    ```bash
    cp .env.example .env

3. Construa e execute o contêiner:

    ``` bash
    docker-compose up --build

4. Acesse no navegador:

    ``` bash
    http://localhost:8501
